Java

Java is a general-purpose computer-programming language that is class-based, object-oriented, and designed specifically to have few implementation dependencies.

Java applications are typically compiled to bytecode that can run on any Java virtual machine (JVM).

As of 2016, Java is one of the most popular programming languages in use.

The Android SDK uses Java as the basis for Android applications.
