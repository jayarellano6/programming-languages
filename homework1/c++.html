C++

C++ is a general-purpose programming language.

Supports programming models such as imperative, object-oriented and generic programming features; also provides for low-level memory manipulation.

C++ has also been found useful in many contexts, most notably: software infrastructure and resource-constrained applications, including desktop applications, servers (e.g. e-commerce, web search or SQL servers).
